    Highlights

    Fastre

    Fastre is a lightweight, fast runtime engine for efficient server-side rendering and dynamic content handling, using HTML-like templating. Ideal for high-performance web applications.

    Autobase

    Autobase is a versatile runtime engine designed for seamless data integration and API middleware. It simplifies how data is handled across different systems.

    Minter

    Minter is a minimal interpreter, a scripting language for making automation simpler and easier. It is written in Java and available on FADE.

    FADE

    FADE is a development environment that includes Fastre and Autobase. It simplifies both frontend web development and backend data integration, making it a complete solution for modern developers.

    Urbanscale

    Urbanscale is an application that evaluates urban environments by providing preset weights for categories such as residential, business, entertainment, and vacation.
